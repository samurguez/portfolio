## Estructura del proyecto
- src/collections/ → datos en JSON (projects, experiences, menu)
- src/pages/ → páginas del sitio (index.astro, about.astro)
- src/components/ → componentes reutilizables
- src/layouts/main.astro → layout base
- public/assets/images/ → imágenes del sitio

## Paleta de colores
- Fondo: oscuro (dark mode por defecto)
- Acento principal: neutral/blanco
- Tipografía: la definida en tailwind.config.mjs

## Reglas de diseño
- Diseño minimalista y profesional
- No usar Inter, Roboto ni fuentes genéricas
- Mantener el estilo visual de la plantilla astro-aria
- Respetar el sistema de dark/light mode existente
- No añadir librerías externas sin consultar primero

## Reglas de desarrollo
- Un cambio a la vez, por secciones
- Hacer git commit antes y después de cada cambio importante
- No modificar tailwind.config.mjs sin consultar
- El contenido está en inglés (V.1) — la versión en español es futura
- Las imágenes de proyectos se actualizarán en la V.1.1
